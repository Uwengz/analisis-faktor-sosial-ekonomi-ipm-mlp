# Analisis Faktor Sosial Ekonomi BPS terhadap IPM menggunakan MLP

## Deskripsi
Proyek ini menganalisis hubungan antara faktor sosial ekonomi dengan 
Indeks Pembangunan Manusia (IPM) menggunakan algoritma MLP Regressor.

## Dataset
Data bersumber dari BPS — 514 kabupaten/kota di Indonesia, 12 variabel.

## Tools & Library
- Python, Pandas, NumPy
- Matplotlib, Seaborn
- Scikit-learn (MLPRegressor, Pipeline, RandomizedSearchCV)

## Hasil
Model MLP tuned mencapai R² = 0.955 pada data uji.  
Faktor dominan: Rata-rata Lama Sekolah, Pengeluaran per Kapita, Umur Harapan Hidup.
